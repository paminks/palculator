# palculator

Palculator is a Free and Open Source Calculator App for GNU/Linux. Written in Qt/C++.


Locate palculator-main in your Terminal and make it executable with 
```
chmod +x ./Palculator
```
And execute with 
```
./Palculator
```

Or make it executable in file properties
